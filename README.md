## About
This Repository is about the first Opening ("Kimi No Sei") and Ending ("Fukashigi No Carte") of the Anime Series "Rascal Does Not Dream Of Bunny Girl Senpai" (Seishun Buta Yarou Wa Bunny Girl Senpai No Yume Wo Minai).
In it you can find the lyrics to both the Opening and Ending aswell as the Animations to the two. 
It provides the lyrics for both the Opening and Ending themes, along with their respective animations.

## Features
- Lyrics: View lyrics clearly to sing along.
- Listen & Watch: Integrated video players allow you to enjoy the audio and animation simultaneously.

## How to use
You can view the website by clicking [here](https://tsukasaxnasa.github.io/Seishun-Buta-Yarou-wa-Bunny-Girl-Senpai-no-Yume-wo-Minai/)

## Dependencies
- Genius: Used to fetch and display song lyrics.
- YouTube: Used for video and audio playback.
